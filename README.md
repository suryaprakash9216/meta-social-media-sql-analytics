# Meta Social Media Analytics using SQL

## Project Overview

This project analyzes a Meta-style social media database using advanced SQL techniques to generate actionable business insights from user activity, content engagement, friendships, groups, pages, advertisements, messaging, and story interactions.

The objective was to simulate real-world business reporting and analytics scenarios commonly performed by Data Analysts, Business Intelligence Analysts, and Product Analysts. Through a series of 15 business-focused SQL problems, the project demonstrates how raw relational data can be transformed into meaningful insights that support decision-making.

---

## Business Objectives

The project focuses on answering key business questions such as:

* Who are the most active users on the platform?
* Which posts generate the highest engagement?
* How strong are user friendship networks?
* Which groups are the most active?
* How effective are advertising campaigns?
* Which users have the highest influence scores?
* What factors contribute to content virality?
* How do users engage across different platform features?
